# Course materials

Kursmaterialien für *Methods of Climate Reconstruction* — getrennt vom Quarto-Analyseprojekt.

## Layout

| Folder | Contents |
|--------|----------|
| `data/` | CRUTEM, PAGES 2.0, N-TREND, DoD2k (`dod2k_v2.0/`), exercise workspaces |
| `exercises/` | `Master_CFR.R`, `Recon_files/`, exercise R Markdown files |
| `materials/` | Additional references and notebooks |

## R project

Open `Methods-of-Climate-Reconstruction.Rproj` in RStudio for the course workflow.

The analysis project in `../proxy-cfr-comparison/` sources `exercises/Master_CFR.R` via relative paths — keep this folder structure intact.
